# Cómo preparo cada repo (estándar profesional)
Para cada repo principal aplica este checklist rápido — copia y pega en el README de cada proyecto:

1. Resumen breve (1-2 líneas).  
2. Capturas / GIF (carpeta `docs/screenshots/`).  
3. Requisitos (SO, versiones, dependencias).  
4. Instalación paso a paso (comandos exactos).  
5. Uso / ejemplos de ejecución.  
6. Tests (si aplican).  
7. Licencia (MIT).  
8. Estado y roadmap (v. futura).  
9. Link a demo / APK / landing.

Ejemplo de bloque "Instalación" (Py):
```bash
git clone https://github.com/mariotc1/CRM_Python.git
cd CRM_Python
python -m venv .venv
source .venv/bin/activate   # macOS / Linux
pip install -r requirements.txt
python main.py
